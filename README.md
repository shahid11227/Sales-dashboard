# Sales-dashboard

Project Title
Sales Performance Dashboard using Power BI

Project Overview
You built an interactive Power BI dashboard to analyze sales and profit performance across regions, categories, states, and years. The dashboard focuses on business level insights and decision support. � Sales_Dashboard-bymir.pdf None

Business Problem
You want to understand where sales come from, which categories perform best, and how profit varies by region and location. The goal is to support data driven decisions.

Dataset Summary
Sales data from 2014 to 2017. Multiple categories including Furniture, Office Supplies, and Technology. Geographic data by region, state, and postal code.

Key KPIs
Total Sales: 2.3 million dollars. Total Profit: 286.40 thousand dollars. Total Orders: 5 thousand. Total Quantity Sold: 38 thousand.

Dashboard Features
Category wise sales contribution with percentage share. Top 10 sub categories by total sales. Year wise sales trend from 2014 to 2017. Region wise sales comparison. Top states by sales performance. Sales versus profit analysis by region and category. Profit distribution by postal code.

Key Insights
Technology leads total sales with about 36.4 percent share. West region generates the highest sales at around 0.73 million dollars. California is the top state by sales with about 0.46 million dollars. Some postal codes show negative profit despite high sales.

Tools Used
Power BI Desktop for data modeling and visualization.

Use Case
You can use this dashboard for sales analysis, regional performance review, and executive reporting.
